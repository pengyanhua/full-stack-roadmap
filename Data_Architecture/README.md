# 数据架构

数据架构是企业数字化转型的基石,涵盖数据建模、治理、管道和平台建设。

## 目录

1. [数据建模](01_data_modeling.md) - 维度建模、范式设计
2. [数据治理](02_data_governance.md) - 质量、血缘、安全
3. [数据管道](03_data_pipeline.md) - ETL/ELT、实时流
4. [数据湖仓](04_data_lake.md) - 数据湖与数据仓库

## 数据架构演进

```
┌────────────────────────────────────────────────────┐
│              现代数据架构                          │
├────────────────────────────────────────────────────┤
│  数据源     业务系统、日志、传感器                 │
│   ↓                                                │
│  数据采集   Kafka、Flume、Logstash                │
│   ↓                                                │
│  数据存储   数据湖(S3/HDFS) + 数据仓库(Snowflake)  │
│   ↓                                                │
│  数据处理   Spark、Flink、dbt                      │
│   ↓                                                │
│  数据服务   BI、ML、API                            │
└────────────────────────────────────────────────────┘
```
