## Getting Started

Each directory contains numbered subdirectories representing the learning sequence:

```
Go/
├── 01-basics/       # Start here
├── 02-functions/
├── 03-structs/
├── 04-concurrency/
├── 05-packages/
├── 06-testing/
├── 07-stdlib/
└── 08-projects/     # End with practical projects
```

Browse to any topic directory and follow the numbered files in order.
