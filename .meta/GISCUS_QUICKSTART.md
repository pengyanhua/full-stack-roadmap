## 🎯 常见问题

### Q1: 安装 Giscus App 后仍显示错误？

**答**：等待 1-2 分钟，GitHub 需要时间同步权限。然后刷新页面。

### Q2: Discussions 在哪里查看？

**答**：访问 https://github.com/pengyanhua/full-stack-roadmap/discussions

所有评论都会显示在这里。

### Q3: 需要更新配置吗？

**答**：如果 Giscus 页面显示 "所有检查已通过"，说明现有配置正确，不需要修改！

只有当 `repo-id` 或 `category-id` 不匹配时才需要更新。

### Q4: 用户必须有 GitHub 账号？

**答**：是的。Giscus 使用 GitHub Discussions，用户需要：
- 登录 GitHub
- 授权 Giscus 应用
- 然后就可以评论

### Q5: 可以管理评论吗？

**答**：可以！在 Discussions 页面你可以：
- 编辑评论
- 删除评论
- 锁定讨论
- 标记为答案
